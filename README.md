# Leaf Disease CNN


This repository contains a Basic CNN model for leaf disease classification with multi-GPU and AMP support, training / evaluation utilities and plotting (confusion matrix + ROC curves).


## Quickstart


1. Put your dataset in `data/augmented_train` and `data/augmented_validation` or change paths in `src/train.py`.
2. Create and activate a virtual environment.
3. Install dependencies: `pip install -r requirements.txt`.
4. Run training: `bash scripts/run_train.sh`.
